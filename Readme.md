# Local Fauna DB Setup

My faunaDB local dev configuration

## Prerequisites

Docker and Docker Compose

## Usage

Clone this repo

Run the `docker-compose`

### ***Example usage***

```
git clone https://github.com/AdityaJain1030/local_faunadb

docker-compose up -d
```
